# Школа Кибергигиены - Интерактивный тест

Проект для студентов СПО 09.02.07 по паспорту "Организация работы школы кибергигиены" [web:17].

## Установка
1. Откройте `index.html` в браузере
2. Тест сохраняет результаты в localStorage

## Функции
- 10 вопросов по фишингу/социнженерии
- Обратная связь с объяснениями
- Прогресс-бар, результаты с ФИО
- Адаптивный дизайн 2026 [web:1]

## Структура
- `style/main.css` - базовые стили
- `style/quiz.css` - стили теста
- `scripts/data.js` - вопросы
- `scripts/quiz.js` - логика
- `scripts/main.js` - инициализация
